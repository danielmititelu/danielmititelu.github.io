# Project: Leave no plant behind

## Parts required
- raspberry pi zero (will work with any raspberry pi)
- 3v-5v pump
- 5v relay
- wires

## Wiring diagram
![image](/pump-scheme.png)

Relay inputs:
- NC = normally opened
- NO = normally closed
- CO = common
- DC- = ground
- DC+ = 5V signal
- IN = signal
